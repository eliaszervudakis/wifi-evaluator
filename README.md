## usage info
```
usage: run.sh [[-h] | [-d] | [-c] | [-r][-a]]
-h: Display this usage info
-d: Do all: capture, read and analyze
-c: Only capture
-r <capture_id>: Read existing capture
-a <capture_id>: Analyze existing read
```
